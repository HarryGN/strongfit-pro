# StrongFit Pro - AI-Powered Fitness Tracker

A comprehensive fitness and nutrition tracking app built with Expo React Native, featuring AI-driven insights and personalized coaching.

## Features

### Core Tracking
- **Weight Management**: Log body weight with trend charts
- **Workout Logging**: Track exercises, weights, reps, and sets with autofill suggestions
- **Nutrition Analysis**: Analyze food photos using Gemini AI for calorie/macronutrient estimates
- **Progress Visualization**: Interactive charts for workout and weight trends

### AI-Powered Features
- **Personalized Encouragement**: Daily motivational messages generated by Gemini AI
- **Smart Nutrition Coach**: AI analysis of food images for detailed nutrition info
- **Intelligent Reminders**: Automated workout notifications based on activity

### User Experience
- **Multi-language Support**: English and Chinese (中文) with easy toggle
- **Tabbed Interface**: Dashboard, Progress, Smart Coach, Nutrition sections
- **Camera Integration**: Direct photo capture for nutrition logging
- **Data Management**: Delete entries, view history in table/chart modes
- **Weekly Goals**: Progress tracking for workout frequency

### Technical Features
- **Local Database**: Expo SQLite for offline data storage
- **Cross-Platform**: iOS and Android support via Expo
- **Standalone Builds**: APK generation with EAS Build
- **Responsive Design**: Optimized for mobile devices

## Setup

### Prerequisites
- Node.js (v18+)
- Expo CLI: `npm install -g @expo/cli`
- EAS CLI: `npm install -g @expo/eas-cli`

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd fitness-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start development server:
   ```bash
   npx expo start
   ```

4. For Android testing:
   - Install Expo Go app
   - Scan QR code from terminal

### Building Standalone APK
1. Login to EAS:
   ```bash
   npx eas login
   ```

2. Build Android APK:
   ```bash
   npx eas build -p android
   ```

3. Download APK from the provided link when build completes.

## Database Schema

### Tables
- **body_weights**: id, date, weight
- **workout_logs**: id, date, exercise_name, weight, reps, sets
- **nutrition_logs**: id, date, food_name, calories, protein, carbs, fat

### Permissions
- Camera: For nutrition photo analysis
- Media Library: For selecting existing photos
- Notifications: For workout reminders

## Configuration

### Environment Variables
Set your Gemini AI API key in the code (currently hardcoded for demo).

### Localization
Default language is Chinese. Toggle via the language button in the app.

## Technologies Used

- **Frontend**: React Native, Expo SDK 54
- **Database**: Expo SQLite
- **AI**: Google Gemini 1.5 Flash
- **Charts**: react-native-chart-kit
- **State Management**: React Context API
- **Build**: EAS (Expo Application Services)
